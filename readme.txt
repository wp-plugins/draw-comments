Donate link: http://www.azettl.de/donate/
Tags: draw, comment, image, drawing
Requires at least: 2.6.2
Tested up to: 2.7.0
Stable tag: 0.0.7

This plugin allows your visitors to draw an image as extra comment.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure you placed `<?php do_action('comment_form', $post->ID); ?>` in your comment templates
